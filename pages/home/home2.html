Reubz Bot

A multifunction Discord Bot.

Extremely easy to use and user friendly.

Moderation, YouTube Alerts, Games, Messages Scheduling and much more!!


Features

Leveling System

Reubz Bot provides with a fully functioning Leveling System for your server. With features like per server rank card, role rewards, leaderboards, and many more configurable features.

Games

This bot allows users to play fun mini-games with other fellow members. Games like tic-tac-toe, rock-paper-scissors, and many more.

YouTube Notifications

Easily set up the bot to send a Notification to a certian channel when a your favourite YouTuber posts a new video.

Message Scheduler

Sending messages automatically at a future date has never been easier.

With a Single command, you can easily create and manage all your scheduled messages.
